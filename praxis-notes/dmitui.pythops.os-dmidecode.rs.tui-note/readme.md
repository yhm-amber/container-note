Thanks to this table, you can retrieve this information without having to probe for the actual hardware. // dmidecode 是一个将计算机的 DMI（有些人说是 SMBIOS）表内容以人类可读格式输出的工具。该表包含系统硬件组件的描述，以及其他有用的信息，如序列号和 BIOS 版本。有了这张表，你就可以在不实际探测硬件的情况下获取这些信息。 /// Why dmitui then ? // 为什么是 dmitui 呢？ /// dmitui is a TUI (Text User Interface) version that allows for easy navigation between sections, unlike dmidecode, which requires you to specify the section as a command-line option.
