To ease deployment and backwards compatibility, 4G and 5G networks can run together in what is known as 5G NR (New Radio) NSA (Non-Standalone) mode. It can only tell you if you are connected to secure or insecure protocols, but it cannot tell you if someone is intercepting information sent over an insecure protocol.
