[src/gh]: https://github.com/jeetsukumaran/Syrupy.git "(GPL-3.0) (Languages: Python 100.0%) System Resource Usage Profiler // 系统资源使用情况分析器 /// Syrupy is a Python script that regularly takes snapshots of the memory and CPU load of one or more running processes, so as to dynamically build up a profile of their usage of system resources. // Syrupy 是一个 Python 脚本，定期对一个或多个正在运行进程的内存和 CPU 负载进行快照，以便动态构建其系统资源使用情况的配置文件。 /// Syrupy supports multiple modes. In the first (default) mode, it monitors the resource usage of a process resulting from the execution of a user-specified command (which can be any arbitrarily-complex combination of programs and arguments that can be invoked from a shell terminal). In the second mode, Syrupy monitors the resource usage of external running processes that meet user-specified criteria: a PID (Process IDentifier) number or a command string matching a regular expression pattern. In the third mode, Syrupy monitors top N processes by memory usage. // Syrupy 支持多种模式。在第一种（默认）模式下，它监控由用户指定命令（可以是任何从 Shell 终端可调用的任意复杂组合的程序和参数）执行所产生的进程的资源使用情况。在第二种模式下，Syrupy 监控符合用户指定条件的外部正在运行进程的资源使用情况：进程 ID（PID）号码或与正则表达式模式匹配的命令字符串。在第三种模式下，Syrupy 按内存使用情况监控前 N 个进程。 /// In either case, the monitoring of the system resource usage is based on repeated calls to the system command “ps”. // 在任何情况下，系统资源使用情况的监控都是基于对系统命令 “ps” 的重复调用。"
