[paper/arxiv]: https://arxiv.org/abs/2510.27643 "(License: arXiv.org perpetual non-exclusive license) [Submitted on 31 Oct 2025] (Cite as: 	arXiv:2510.27643 [stat.ML]) (arXiv-issued DOI via DataCite: 	doi.org/10.48550/arXiv.2510.27643) { Machine Learning (stat.ML); Machine Learning (cs.LG); Numerical Analysis (math.NA); Optimization and Control (math.OC); Computation (stat.CO) } Bayesian Optimization on Networks // 网络贝叶斯优化 /// This paper studies optimization on networks modeled as metric graphs. Motivated by applications where the objective function is expensive to evaluate or only available as a black box, we develop Bayesian optimization algorithms that sequentially update a Gaussian process surrogate model of the objective to guide the acquisition of query points. To ensure that the surrogates are tailored to the network's geometry, we adopt Whittle-Matérn Gaussian process prior models defined via stochastic partial differential equations on metric graphs. In addition to establishing regret bounds for optimizing sufficiently smooth objective functions, we analyze the practical case in which the smoothness of the objective is unknown and the Whittle-Matérn prior is represented using finite elements. Numerical results demonstrate the effectiveness of our algorithms for optimizing benchmark objective functions on a synthetic metric graph and for Bayesian inversion via maximum a posteriori estimation on a telecommunication network. // 本文研究以度量图为模型的网络优化问题。受目标函数评估成本高昂或仅以黑盒形式可用的应用场景启发，我们开发了贝叶斯优化算法，该算法通过依次更新目标函数的高斯过程代理模型来指导查询点的获取。为确保代理模型适应网络的几何结构，我们采用在度量图上通过随机偏微分方程定义的 Whittle-Matérn 高斯过程先验模型。除了为优化充分光滑的目标函数建立遗憾界限外，我们还分析了目标函数的光滑性未知且 Whittle-Matérn 先验使用有限元表示的实际情况。数值结果表明，我们的算法在合成度量图上优化基准目标函数以及通过最大后验估计在电信网络上进行贝叶斯反演方面具有有效性。"
