[games/site]: https://lutris.net/games/ "Preparing your computer for Lutris // 为 Lutris 准备您的计算机 /// Lutris is an open gaming platform for Linux. Lutris helps you install and play video games from all eras and from most gaming systems. By leveraging and combining existing emulators, engine re-implementations and compatibility layers, it gives you a central interface to launch all your games. The client can connect with existing services like Humble Bundle, GOG and Steam to make your game libraries easily available. Game downloads and installations are automated and can be modified through user made scripts. Before installing Lutris (or at least before running any games), make sure your computer is fully setup to run games. This includes making sure your graphics drivers are installed and up to date, that Vulkan is installed and that the 32bit libraries for OpenGL / Vulkan are present. If any of those components are missing Lutris will give a warning. We also very highly recommend installing Wine from your package manager. This can be any version of Wine (stable, development or staging) as long as it pulls both 32bit and 64bit variants. Lutris will usually not make use of it directly but the peer dependencies installed from the Wine package are necessary to run our Wine version properly. // Lutris 是一个开源的 Linux 游戏平台。Lutris 帮助您安装和游玩来自各个时代以及大多数游戏系统的视频游戏。通过利用和结合现有的模拟器、引擎重实现和兼容层，它为您提供了一个中央界面来启动所有您的游戏。客户端可以连接到现有的服务，如 Humble Bundle、GOG 和 Steam，使您的游戏库轻松可用。游戏下载和安装是自动化的，并且可以通过用户制作的脚本进行修改。在安装 Lutris（或者至少在运行任何游戏之前），请确保您的计算机已完全设置好以运行游戏。这包括确保您的图形驱动程序已安装并更新，Vulkan 已安装，并且存在 OpenGL / Vulkan 的 32 位库。如果这些组件中缺少任何一个，Lutris 将会给出警告。我们还强烈建议您通过包管理器安装 Wine。这可以是任何版本的 Wine（稳定版、开发版或预发布版），只要它能够拉取 32 位和 64 位的变体即可。 Lutris 通常不会直接使用它，但从 Wine 软件包安装的同伴依赖项对于正确运行我们的 Wine 版本是必要的。"
