[src/gh]: https://github.com/josephvoss/git-remote-s3.git "(MIT) (Languages: Rust 99.5%, Makefile 0.5%) Git remote helper (. web: git-scm.com/docs/gitremote-helpers) to communicate to an S3 backend directly. Able to fetch and push git objects to S3-compatible object stores. // 用于直接与 S3 后端通信的 Git 远程助手 。能够从 S3 兼容的对象存储中获取和推送 Git 对象。 /// Heavily leverages gitoxide (. gh: Byron/gitoxide.git) and rust-s3 (. crates: rust-s3) to do the heavy lifting talking to the respective backends.
