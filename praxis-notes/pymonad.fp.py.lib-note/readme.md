[src/gh]: https://github.com/jasondelaat/pymonad.git "(BSD-3-Clause) (Languages: Python 100.0%) PyMonad implements data structures typically available in pure functional or functional first programming languages like Haskell and F#. Included are Monad and Monoid data types with several common monads included - such as Maybe and State - as well as some useful tools such as the @curry decorator for defining curried functions. PyMonad 2.0.x represents and almost complete re-write of the library with a simpler, more consistent interface as well as type annotations to help ensure correct usage. // PyMonad 实现了通常可用的纯数据结构 函数式或函数式优先编程语言，例如 Haskell 和 F＃。 包括 Monad 和 Monoid 数据类型以及几种常见的 包含 monad - 例如 Maybe 和 State - 以及一些有用的 用于定义柯里化的工具，例如 @curry 装饰器 功能。 PyMonad 2.0.x 代表并几乎完全重写了 该库具有更简单、更一致的接口和类型 注释以帮助确保正确使用。"
