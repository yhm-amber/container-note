The API consists of two main components: a plugin API which integrates external functionality into the CTP, and a JSON API which allows CTP functionality to be integrated into external sites. // 中文文本项目应用程序编程接口（CTP API）提供集成中文文本项目内容和功能到其他基于网页或能感知网页的站点和应用程序的方法。该 API 由两个主要组件构成：一个插件 API，用于将外部功能集成到中文文本项目中，以及一个 JSON API，允许中文文本项目的功能被集成到外部站点。 /// This page contains technical documentation for those interested in creating their own plugins.
