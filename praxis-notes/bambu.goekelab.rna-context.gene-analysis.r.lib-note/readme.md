[intro.article/.site]: https://goekelab.github.io/bambu/articles/bambu.html "Bambu is a method for transcript discovery and quantification from long read RNA-Seq data. Bambu uses aligned reads and genome reference annotations as input, and will return abundance estimates for all known transcripts and for newly discovered transcripts. Bambu uses the information from the reference annotations to correct misalignment at splice junctions, then reduces the aligned reads to read equivalent classes, and uses this information to identify novel transcripts across all samples of interest. Reads are then assigned to transcripts, and expression estimates are obtained using an expectation maximisation algorithm. Here, we present an example workflow for analysing Nanopore long read RNA-Sequencing data from two human cancer cell lines from the Singapore Nanopore Expression Project (SG-NEx). // Bambu 是一种利用长读长 RNA-Seq 数据进行转录本发现和定量的方法。Bambu 以比对后的 reads 和基因组参考注释作为输入，并返回所有已知转录本和新发现转录本的丰度估计值。Bambu 利用参考注释信息校正剪接位点的错配，然后将比对后的 reads 归类为等效的 read 类型，并利用这些信息在所有目标样本中识别新的转录本。随后，将 reads 分配给相应的转录本，并使用期望最大化算法获得表达量估计值。本文展示了一个分析新加坡纳米孔表达项目 (SG-NEx) 中两个人类癌细胞系纳米孔长读长 RNA 测序数据的示例工作流程。"
