Today TWRP is the leading custom recovery for Android phones. // Team Win 最初是为了将 WiMAX 移植到 CM7 以支持 HTC EVO 4G 而成立的。在完成 EVO 4G 的项目后，我们希望开发一个能够支持更多设备的项目，最终决定专注于开发一个恢复系统。如今 TWRP 已成为 Android 手机领域领先的定制恢复系统。 /// A custom recovery is used for installing custom software on your device.
