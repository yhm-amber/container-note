[site/ghio]: https://simonpcouch.github.io/gander/ "gander // 观鸟 ///: gander is a higher-performance and lower-friction chat experience for data scientists in RStudio and Positron–sort of like completions with Copilot, but it knows how to talk to the objects in your R environment. The package brings ellmer chats into your project sessions, automatically incorporating relevant context and streaming their responses directly into your documents. // gander 是为 RStudio 和 Positron 中的数据科学家提供的更高性能、更低摩擦的聊天体验——有点像使用 Copilot 的补全功能，但它知道如何与您 R 环境中的对象进行交流。该软件包将 ellmer 聊天引入您的项目会话中，自动整合相关上下文，并将它们的回复直接流式传输到您的文档中。 /// Why not just chat? In many ways, working with gander is just like using a chat interface online or via shinychat. The gander assistant will automatically find the context it needs, though: // 为什么不直接聊天？在很多方面，使用 gander 就像在线或通过 shinychat 使用聊天界面一样。不过，gander 助手会自动找到它所需的上下文，尽管： /// - Code context: File contents from elsewhere in the project you’re working on (e.g. the lines in your source file). // 代码上下文：您正在处理的项目中其他位置文件的内容（例如，您的源文件中的行）。 /// - Environment context: Variables, allowing the assistant to locate the column names and types in data frames you’re working with, images linked to in your documents, and function definitions. This is what differentiates gander from many other assistants out there; gander can interface with your R session to describe your computational environment. // 环境上下文：变量，允许助手定位你在处理的数据框中的列名和类型、文档中链接的图像以及函数定义。这是 gander 与许多其他助手不同的地方；gander 可以与你的 R 会话接口，描述你的计算环境。 /// Important // 重要 ///: For now, gander is particularly effective in .R files but struggles to provide appropriately formatted responses in .Qmd and related file formats. // 目前，gander 在 .R 文件中特别有效，但在 .Qmd 和相关文件格式中难以提供适当格式的响应。"
