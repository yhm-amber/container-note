[paper/arxiv]: https://arxiv.org/abs/1911.03599 "[Submitted on 9 Nov 2019] { Computer Vision and Pattern Recognition (cs.CV) } (doi: 10.48550/arXiv.1911.03599) CenterFace: Joint Face Detection and Alignment Using Face as Point // CenterFace：以人脸为点的联合人脸检测与对齐 /// Face detection and alignment in unconstrained environment is always deployed on edge devices which have limited memory storage and low computing power. This paper proposes a one-stage method named CenterFace to simultaneously predict facial box and landmark location with real-time speed and high accuracy. The proposed method also belongs to the anchor free category. This is achieved by: (a) learning face existing possibility by the semantic maps, (b) learning bounding box, offsets and five landmarks for each position that potentially contains a face. Specifically, the method can run in real-time on a single CPU core and 200 FPS using NVIDIA 2080TI for VGA-resolution images, and can simultaneously achieve superior accuracy (WIDER FACE Val/Test-Easy: 0.935/0.932, Medium: 0.924/0.921, Hard: 0.875/0.873 and FDDB discontinuous: 0.980, continuous: 0.732). // 在非约束环境下进行人脸检测和对齐通常部署在内存有限、计算能力较弱的边缘设备上。本文提出了一种名为 CenterFace 的单阶段方法，能够实时、高精度地同时预测人脸边界框和关键点位置。该方法属于无锚框方法。其实现方式为：(a) 通过语义图学习人脸存在的可能性；(b) 学习每个可能包含人脸的位置的边界框、偏移量和五个关键点。具体而言，该方法在 NVIDIA 2080TI 显卡上处理 VGA 分辨率图像时，可在单核 CPU 上以 200 FPS 的速度实时运行，并同时获得优异的精度（WIDER FACE 验证集/测试集 - 简单：0.935/0.932，中等：0.924/0.921，困难：0.875/0.873；FDDB 不连续：0.980，连续：0.732）。"
[knowsby]: https://github.com/ORB-HD/deface.git "How it works // 工作原理 /// The included face detection system is based on CenterFace (code (. gh: Star-Clouds/CenterFace.git), paper (. arxiv: 1911.03599)), a deep neural network optimized for fast but reliable detection of human faces in photos. The network was trained on the WIDER FACE (. web: shuoyang1213.me/WIDERFACE) dataset, which contains annotated photos showing faces in a wide variety of scales, poses and occlusions. // 该系统包含的人脸检测功能基于 CenterFace（ 代码 ， 论文 ），这是一个深度神经网络，专为快速可靠地检测照片中的人脸而优化。该网络使用 WIDER FACE 数据集进行训练，该数据集包含标注照片，展示了各种尺度、姿态和遮挡情况下的人脸。 /// Although the face detector is originally intended to be used for normal 2D images, deface can also use it to detect faces in video data by analyzing each video frame independently. The face bounding boxes predicted by the CenterFace detector are then used as masks to determine where to apply anonymization filters. // 尽管人脸检测器最初是为处理普通二维图像而设计的，但 deface 也可以通过独立分析每个视频帧来检测视频数据中的人脸。CenterFace 检测器预测的人脸边界框随后被用作掩码，以确定匿名化滤波器的应用位置。"
