Write and manage journal/notes entries from the terminal. // 从终端编写和管理日记/笔记条目。 /// - Store your entries in either a plain text file using the JSON format or a SQLite database. // 将您的条目存储在纯文本文件（使用 JSON 格式）或 SQLite 数据库中。 /// -
