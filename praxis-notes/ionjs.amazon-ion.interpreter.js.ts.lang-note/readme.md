The text format (a superset of JSON) is easy to read and author, supporting rapid prototyping. The binary representation is efficient to store, transmit, and skip-scan parse. The rich type system provides unambiguous semantics for long-term preservation of data which can survive multiple generations of software evolution. // Amazon Ion 是一种丰富类型、自我描述、分层数据序列化格式，提供可互换的二进制和文本表示形式。文本格式（JSON 的超集）易于阅读和编写，支持快速原型设计。二进制表示形式在存储、传输和跳过解析方面高效。丰富的类型系统为长期保存数据提供了明确的语义，这些数据能够经受多代软件演进的考验。 /// Ion was built to address rapid development, decoupling, and efficiency challenges faced every day while engineering large-scale, service-oriented architectures.
