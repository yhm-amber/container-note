[lib.src/gh]: https://github.com/vishpat/lisp-rs.git "(MIT) (Languages: Rust 99.9%, Shell 0.1%) A simple Lisp interpreter/library in Rust. The interpreter/library was initially developed as a teaching aid to explain how Lisp interpreters work and how can they be implemented using the Rust programming language. However the interpreter seems to be taking life of it's own and has been ported to the web using WASM. The interpreter is available as a crate and can be used to embed a Lisp interpreter in your Rust projects. The WASM implementation uses the lisp-rs as a library to implement the online interpreter. // Rust 中的一个简单的 Lisp 解释器/库。该解释器/库最初是作为教学辅助工具开发的，用于解释 Lisp 解释器如何工作以及如何使用 Rust 编程语言实现它们。然而，解释器似乎正在夺走它自己的生命，并已使用 WASM 移植到网络上。该解释器以 crate 的形式提供，可用于将 Lisp 解释器嵌入到您的 Rust 项目中。 WASM 实现使用 lisp-rs 作为库来实现在线解释器。"
