To exit niri, press SuperShiftE. // 安装完成后，从你的显示管理器（如 GDM）启动 niri。按 Super T 运行终端（Alacritty），按 Super D 运行应用启动器（fuzzel）。要退出 niri，按 Super Shift E 。 /// If you're not using a display manager, you should run niri-session (systemd/dinit) or niri --session (others) from a TTY. The --session flag will make niri import its environment variables globally into the system manager and D-Bus, and start its D-Bus services. The niri-session script will additionally start niri as a systemd/dinit service, which starts up a graphical session target required by some services like portals. // 如果你没有使用显示管理器，你应该从 TTY 运行 niri-session （systemd/dinit）或 niri --session （其他）。 --session 标志将使 niri 将其环境变量全局导入系统管理器和 D-Bus，并启动其 D-Bus 服务。 niri-session 脚本将额外以 systemd/dinit 服务的形式启动 niri，该服务启动了某些服务（如门户）所需的图形会话目标。 /// You can also run niri inside an existing desktop session.
