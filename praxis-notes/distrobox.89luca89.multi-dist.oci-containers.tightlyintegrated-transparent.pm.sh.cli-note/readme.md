distrobox-assemble – create and destroy containers based on a config file // distrobox-assemble – 根据配置文件创建和销毁容器 /// - distrobox-create – create a container // distrobox-create – 创建容器 /// - distrobox-enter – enter a container // distrobox-enter – 进入容器 /// - distrobox-ephemeral – create a temporal container, destroy it when exiting the shell // distrobox-ephemeral – 创建一个临时容器，并在退出 shell 时销毁它 /// - distrobox-list – list containers created with distrobox // distrobox-list – 列出使用 distrobox 创建的容器 /// - distrobox-rm – delete a container created with distrobox // distrobox-rm – 删除使用 distrobox 创建的容器 /// - distrobox-stop – stop a running container created with distrobox // distrobox-stop – 停止使用 distrobox 创建的正在运行的容器 /// - distrobox-upgrade – upgrade one or more running containers created with distrobox at once // distrobox-upgrade – 一次性升级一个或多个使用 distrobox 创建的正在运行的容器 /// - distrobox-generate-entry – create an entry of a created container in the applications list // distrobox-generate-entry – 在应用程序列表中创建已创建容器的条目 /// - distrobox-init – entry point of the container (not meant to be used manually) // distrobox-init – 容器的入口点（不建议手动使用） /// - distrobox-export – use inside the container, export apps and services from the container to the host // distrobox-export – 在容器内使用，将应用程序和服务从容器导出到主机 /// - distrobox-host-exec – run commands/programs from the host, while inside of the container // distrobox-host-exec – 在容器内部运行来自主机的命令/程序 /// It also includes a little wrapper to launch commands with distrobox COMMAND instead of calling the single files. // 它还包含一个用于通过 distrobox COMMAND 启动命令的小型包装器 而不是调用单个文件。"
