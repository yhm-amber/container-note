FUSE-Rust provides a replacement for the libfuse userspace library between these two. This way, a developer can fully take advantage of the Rust type interface and runtime features when building a FUSE filesystem in Rust. // 内核驱动程序由 FUSE 项目提供，用户空间实现则需要开发者自行完成。FUSE-Rust 为这两者之间提供了一个 libfuse 用户空间库的替代方案。这样，开发者在用 Rust 构建 FUSE 文件系统时，就可以充分利用 Rust 的类型接口和运行时特性。 /// Except for a single setup (mount) function call and a final teardown (umount) function call to libfuse, everything runs in Rust, and on Linux these calls to libfuse are optional.
