[core.src/gh]: https://github.com/RohanAdwankar/oxdraw.git "(MIT) (Languages: Rust 57.5%, TypeScript 38.3%, CSS 3.3%, Other 0.9%) Diagram as Code Tool Written in Rust with Draggable Editing // 使用 Rust 编写的可拖动编辑的图示代码工具 /// The goal of oxdraw is to make it easy to create and maintain high-quality diagrams using a declarative and reproducible syntax. Charts are written in Mermaid syntax, while a web interface allows users to fine-tune positions connector paths, colors, and other styling components. Whenever a diagram is tweaked visually, the structural changes are persisted back to the source file as declarative code so that everything remains deterministic and versionable. The changes are saved as comments in the mermaid file so it remains compatible with other Mermaid tools. The repo is composed of the Rust CLI to compile .mmd files into images and the React based web interface to editing the files. // oxdraw 的目标是使用声明性和可重复的语法轻松创建和维护高质量的图表。图表使用 Mermaid 语法编写，同时提供了一个 Web 界面，允许用户微调连接器路径、颜色和其他样式组件。每当图表在视觉上进行调整时，结构变化会作为声明性代码持久化回源文件，以便所有内容都保持确定性和可版本控制。更改会作为注释保存在 mermaid 文件中，以保持与其他 Mermaid 工具的兼容性。该存储库由用于将 .mmd 文件编译为图像的 Rust CLI 和基于 React 的 Web 界面组成用于编辑文件。"
