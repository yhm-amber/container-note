It does not know (nor does it need to know) the rules of the game you are playing. Instead, it lets you and your friends play with common tabletop/boardgame elements such as hands of cards, boards, and so on. // 但它与大多数在线游戏系统不同。它不知道（也不需要知道）你正在玩的游戏规则。相反，它允许你和你的朋友使用常见的桌面/棋盘游戏元素，例如手牌、棋盘等等。 /// So it's something like a “tabletop simulator” (but it does not have any 3D, or a physics engine, or anything like that). /// 所以它有点像“桌面模拟器”（但它没有任何 3D 效果、物理引擎或类似的东西）。 /// This means that with Otter: // 这意味着使用 Otter： /// - Supporting a new game, that Otter doesn't know about yet, would usually not involve writing or modifying any computer programs. // 支持 Otter 还不知道的新游戏，通常不会涉及编写或修改任何计算机程序。 /// - If Otter already has the necessary game elements (cards, say) all you need to do is write a spec file saying what should be on the table at the start of the game. For example, most Whist variants that start with a standard pack of 52 cards are already playable. // 如果 Otter 已经具备了必要的游戏元素（例如卡牌），你只需要编写一个规范文件，说明游戏开始时桌面上应该摆放哪些东西。例如，大多数使用标准 52 张牌的惠斯特牌变体游戏都可以直接进行游戏。 /// - You can play games where the rules change as the game goes along, or are made up by the players, or are too complicated to write as a computer program. // 你可以玩一些游戏，这些游戏的规则会随着游戏的进行而改变，或者由玩家制定，或者过于复杂而无法用计算机程序编写。 /// - House rules are no problem, since the computer isn't enforcing the rules - you and your friends are.家规不是问题，因为电脑不会强制执行这些规则——而是由你和你的朋友们来执行。 /// - Everyone can interact with different items on the game table, at any time. (Otter doesn't know about your game's turn-taking, so doesn't know whose turn it might be.) // 游戏桌上的每个人都可以随时与不同的物品互动。（Otter 不知道你们游戏的轮流进行规则，所以不知道现在轮到谁了。） /// We have played successful and fun online games of both Penultima and Mao with Otter. // 我们和 Otter 一起玩过 Penultima 和 Mao 的在线游戏，都玩得很开心，也很成功。 /// Playing a game with Otter // 和水獭一起玩游戏 /// The Otter game environment is accessed from a web browser running JavaScript, using a magic https link obtained from joining the game. // 通过运行 JavaScript 的网络浏览器访问 Otter 游戏环境，使用加入游戏后获得的魔法 https 链接。 /// You will need to be able to talk to your friends about the game, while you play. Otter works well when used together with a voice chat - we have had success with Jitsi in voice-only mode. // 玩游戏的时候，你需要能够和朋友们交流。Otter 搭配语音聊天软件使用效果很好——我们用 Jitsi 的纯语音模式也取得了不错的效果。 /// Most relatively modern desktop browsers should be able to work with Otter. (The most advanced feature needed is support for WebAssembly.) // 大多数较新的桌面浏览器应该都能与 Otter 兼容。（所需的最高级功能是支持 WebAssembly。） /// Predefined games and pieces currently available // 目前可用的预定义游戏和棋子 /// Otter currently has, in its pieces library:Otter 目前在其组件库中包含： /// - Ingredients for chess, including fairy chess. So there's a board, pieces, including many fairy chess pieces, and a chess clock. // 下棋所需的材料，包括仙女棋。所以这里有棋盘、棋子（包括许多仙女棋子）和一个棋钟。 /// - Ingredients for card games. So, a deck of 52 standard playing cards, plus two kinds of joker. Special “hand” and “deck” pieces for player hands and a pickup deck. // 纸牌游戏所需材料：一副52张的标准扑克牌，外加两种大小王。玩家手牌和副牌的专用“手牌”和“牌堆”配件，以及一副备用牌。 /// Currently there are game definitions for: // 目前有以下游戏定义： /// - Penultima. This can be used directly to play standard chess and some fairy chess variants. // 倒数第二张棋子。它可以直接用于下标准国际象棋和一些仙女国际象棋的变体。 /// - Mao. This can be used to play any game of roughly that shape. // 毛式。这可以用来玩任何形状大致相同的游戏。 /// Defining new games using the existing pieces from the library is fairly easy. It is also possible to add elements from the library ad-hoc, even while a game is in progress. // 使用库中现有的组件创建新游戏相当容易。即使在游戏进行中，也可以随时从库中添加组件。 /// Limitations // 局限性 /// Currently, joining a game requires a unix shell account on the server host (or help from a shell account user). // 目前，加入游戏需要服务器主机上的 unix shell 帐户（或 shell 帐户用户的帮助）。 /// There is not currently a publicly available server. The server code is Free Software and if you have a suitable vm or server you are encouraged to build and run it yourself, for you and your friends. // 目前没有公开可用的服务器。服务器代​​码是自由软件，如果您有合适的虚拟机或服务器，我们鼓励您自行搭建并运行，供您和您的朋友使用。 /// Mobile phones are not really suitable for playing on Otter because their screens are too small. Tablets and other touchscreen based systems could be made to work, but don't work well right now. // 手机屏幕太小，不太适合在 Otter 系统上玩游戏。平板电脑和其他触屏设备虽然也能玩，但目前体验并不理想。 /// Otter does not currently have even a built-in text chat facility. It does have a way to share a URL for a voice chat. // Otter 目前甚至没有内置的文字聊天功能。它确实提供了一种分享语音聊天 URL 的方法。 /// Free software, and user freedom // 自由软件和用户自由 /// Otter is Free Software. I wrote it to liberate game players from the need to encode their game rules as computer programs and thus from the tyranny of programmers :-). // Otter 是自由软件。我编写它是为了让游戏玩家摆脱将游戏规则编码成计算机程序的需要，从而摆脱程序员的束缚 :-) 。 /// I would love contributions, particularly to address the limitations I mention above, and to improve the user experience. // 我非常欢迎大家贡献力量，特别是针对我上面提到的局限性，以及改善用户体验。 /// I am also working to make it possible to let users define their own games (including their own pieces, cards, boards, and so on) without having to install them on the server. // 我目前也在努力让用户能够自定义游戏（包括自定义棋子、卡片、棋盘等），而无需将它们安装到服务器上。"
