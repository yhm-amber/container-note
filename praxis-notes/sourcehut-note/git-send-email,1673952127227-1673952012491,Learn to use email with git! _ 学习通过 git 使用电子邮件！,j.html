"Signing off" on your commits“签署”你的承诺

Some projects, such as the Linux kernel, will ask you to "sign off" on your commits. To do this, add --signoff (or -s) to git send-email. To set it as the default for that git repository: 一些项目，例如 Linux 内核，会要求您“签署”您的提交。为此，将 --signoff（或 -s）添加到 git send-email。要将其设置为该 git 存储库的默认值：

The meaning of a signoff depends on the project to which you’re committing.
