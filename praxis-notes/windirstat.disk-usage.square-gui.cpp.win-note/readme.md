[site/net]: https://windirstat.net/ "The treemap represents each file as a colored rectangle, the area of which is proportional to the file's size. The rectangles are arranged in such a way, that directories again make up rectangles, which contain all their files and subdirectories. So their area is proportional to the size of the subtrees. The color of a rectangle indicates the type of the file, as shown in the extension list. The cushion shading additionally brings out the directory structure. // 树状图将每个文件表示为一个彩色矩形，其面积与文件大小成正比。矩形排列的方式是，目录再次构成包含其所有文件和子目录的矩形。因此，它们的面积与子树的大小成正比。矩形的颜色表示文件类型，如扩展名列表所示。垫片阴影进一步突出了目录结构。"
