It provides an efficient way to combine and view disaggregated results, while also creating publication ready figures and data tables. // Marmot 是一款用于生产成本和产能扩张建模结果的数据格式化和可视化工具。它提供了一种高效的方式来合并和查看细分结果，同时还能生成可用于出版的图表和数据表。 /// Marmot: Energy Analysis and Visualization // Marmot：能量分析和可视化 /// What is it? // 那是什么？ /// Marmot is a data formatting and visualization tool for production cost and capacity expansion modelling results. It provides an efficient way to analysis data by combing temporally disaggregated results and allowing the aggregation of different device types and modelling regions. // Marmot 是一款用于生产成本和产能扩张建模结果的数据格式化和可视化工具。它通过整合按时间顺序分类的结果，并允许汇总不同设备类型和建模区域，提供了一种高效的数据分析方法。 /// Marmot currently supports analysis of PLEXOS (. web: energyexemplar.com/plexos) production costs modelling and ReEDS (. web: nrel.gov/analysis/reeds) capacity expansion results. // Marmot 目前支持对 PLEXOS 生产成本模型和 ReEDS 产能扩张结果进行分析。 /// Main Features // 主要特点 /// - Formats modelling results data to a standard format and saves them to a hdf5 file. // 将建模结果数据格式化为标准格式，并将其保存到 hdf5 文件中。 /// - Combines temporally disaggregated results. // 整合了按时间顺序细分的结果。 /// - Handles every timestep whether that be 5 minute or yearly interval data. // 能够处理所有时间步长的数据，无论是 5 分钟间隔还是每年间隔的数据。 /// - Provides approximately 120 pre-built plots which offer vast user customization through an easy to use configuration file and various mapping and input data csvs. // 提供约 120 个预置图表，可通过易于使用的配置文件和各种映射和输入数据 csv 文件提供广泛的用户自定义功能。"
