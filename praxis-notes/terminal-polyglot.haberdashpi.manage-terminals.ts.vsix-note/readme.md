[src/gh]: https://github.com/haberdashPI/terminal-polyglot.git "(MIT) (Languages: TypeScript 100.0%) Easily manage terminals on a per-language basis in VSCode // 在 VSCode 中轻松地按语言管理终端。 /// Terminal Polyglot // 终端多语者 /// This extension provides a set of commands for sending text to an integrated terminal specific to the programming language of the current file (or code fence). It associates each file with a language-specific terminal instance: any command you issue through the extension is sent to that terminal. // 此扩展提供了一组命令，用于将文本发送到与当前文件（或代码块）编程语言相关的集成终端。它将每个文件与特定语言的终端实例关联起来：您通过此扩展发出的任何命令都会发送到该终端。 /// The extension allows for multiple terminals per language, and persistent terminal sessions via tmux or screen. If multiple terminals are present for a given language, text sent from a file will be sent to the last terminal used for that file. // 该扩展允许每种语言使用多个终端，并通过 tmux 或 screen 实现持久的终端会话。如果某种语言存在多个终端，则从文件中发送的文本将发送到最后用于访问该文件的终端。 /// The commands include: // 命令包括： /// - Open an existing terminal specific to the language of the current file or code fence (default key Ctrl+'). If no terminal exists for that language, a new one is created. // 打开一个与当前文件或代码块语言对应的终端（默认快捷键 Ctrl+' ）。如果不存在该语言的终端，则会创建一个新的终端。 /// - Open a new terminal specific to the language of the current file (default key Ctrl+Shift'). A new REPL is launched. // 打开一个与当前文件语言对应的新终端（默认快捷键 Ctrl+Shift' ）。一个新的 REPL 程序将启动。 /// - Cycle through all terminals for a specific language (command Terminal Polyglot: Next Terminal and Termianl Polyglot: Previous Terminal) // 循环切换特定语言的所有终端（命令 Terminal Polyglot: Next Terminal 和 Termianl Polyglot: Previous Terminal ） /// - Open the Nth language-specific terminal, creating it if it does not exist (command Terminal Polyglot: Open Terminal...) // 打开第 N 个特定语言的终端，如果不存在则创建它（命令 Terminal Polyglot: Open Terminal... ） /// - Send lines of code to the language-specific terminal (default key Ctrl+Shift+Enter), opening the terminal and launching a REPL for that language if necessary. // 将代码行发送到特定语言的终端（默认快捷键 Ctrl+Shift+Enter ），必要时打开终端并启动该语言的 REPL。 /// - Send a command to change the directory to the current file's location (command Terminal Polyglot: Change Directory) // 发送命令将目录更改为当前文件所在位置（命令 Terminal Polyglot: Change Directory ） /// - Send a command to run the entire file (command Terminal Polyglot: Run File). // 发送命令运行整个文件（命令 Terminal Polyglot: Run File ）。 /// - Change directory to file location in a shell (command Terminal Polyglot: Change Shell Directory) // 在 shell 中更改文件位置目录（命令 Terminal Polyglot: Change Shell Directory ） /// - Send / select code inside a code fence // 在代码围栏内发送/选择代码 /// - Move to / select the next / previous code fence // 移动到/选择下一个/上一个代码围栏 /// - Send an arbitrary command to the terminal // 向终端发送任意命令"
