[src/gh]: https://github.com/Bubka/2FAuth.git "(AGPL-3.0) (Languages: PHP 89.3%, Vue 8.5%, RobotFramework 2.2%) A Web app to manage your Two-Factor Authentication (2FA) accounts and generate their security codes // 一款用于管理您的双因素身份验证 (2FA) 帐户并生成其安全代码的 Web 应用程序。 /// Purpose // 目的 /// 2FAuth is a web based self-hosted alternative to One Time Passcode (OTP) generators like Google Authenticator, designed for both mobile and desktop. // 2FAuth 是一个基于 Web 的自托管一次性密码 (OTP) 生成器，类似于 Google Authenticator，专为移动设备和桌面设备设计。 /// It aims to ease you perform your 2FA authentication steps whatever the device you handle, with a clean and suitable interface. // 它旨在通过简洁易用的界面，让您无论使用何种设备，都能轻松完成 2FA 身份验证步骤。 /// I created it because : // 我创建它的原因是： /// - Most of the UIs for this kind of apps show tokens for all accounts in the same time with stressful countdowns (in my opinion) // 在我看来，这类应用程序的大多数用户界面都会同时显示所有帐户的令牌，并配以令人紧张的倒计时。 /// - I wanted my 2FA accounts to be stored in a standalone database I can easily backup and restore (did you already encountered a smartphone loss with all your 2FA accounts in Google Auth? I did...) // 我希望我的双重验证账户存储在一个独立的数据库中，这样我可以轻松地备份和恢复（你是否曾经因为丢失手机而丢失了所有存储在 Google Auth 中的双重验证账户？我经历过……） /// - I hate taking out my smartphone to get an OTP when I use a desktop computer // 我用台式电脑的时候，最讨厌的就是拿出手机去获取验证码。 /// - I love coding and I love self-hosted solutions // 我热爱编程，也热爱自托管解决方案。 /// Main features // 主要特点 /// - Manage your 2FA accounts and organize them using Groups // 管理您的双因素认证帐户，并使用群组进行组织 /// - Scan and decode any QR code to add account in no time // 扫描并解码任何二维码，即可快速添加账户 /// - Add custom account without QR code thanks to an advanced form // 借助高级表单，无需二维码即可添加自定义帐户。 /// - Edit accounts, even the imported ones // 编辑帐户，包括导入的帐户。 /// - Generate TOTP and HOTP security codes and Steam Guard codes // 生成 TOTP 和 HOTP 安全码以及 Steam Guard 代码 /// 2FAuth is currently fully localized in English and French. See Contributing if you want to help on adding more languages. // 2FAuth 目前已完全本地化为英语和法语。如果您想帮助添加更多语言，请参阅 “贡献”部分 。 /// Security // 安全 /// 2FAuth provides several security mechanisms to protect your 2FA data as best as possible. // 2FAuth 提供多种安全机制，以尽可能地保护您的双因素身份验证数据。 /// - Single user app // 单用户应用 ///: You have to create a user account and authenticate yourself to use the app. It is not possible to create more than one user account, the app is thought for personal use. // 您必须创建用户帐户并进行身份验证才能使用此应用。您只能创建单个用户帐户，此应用仅供个人使用。 /// - Modern authentication // 现代身份验证 ///: You can sign in 2FAuth using a security key like a Yubikey or a Titan key and disable the traditional login form. // 您可以使用 Yubikey 或 Titan 密钥等安全密钥登录 2FAuth，并禁用传统的登录表单。 /// - Data encryption // 数据加密 ///: Sensitive data stored in the database can be encrypted to protect them against db compromise. Encryption is provided as an option which is disabled by default. It is strongly recommended to backup the APP_KEY value of your .env file (or the whole file) when encryption is On. // 数据库中存储的敏感数据可以加密，以防止数据库遭到入侵。加密功能默认处于禁用状态，因此是一个可选功能。强烈建议在启用加密时备份 .env 文件中的 APP_KEY 值（或整个文件）。 /// - Auto logout // 自动登出 ///: 2FAuth automatically log you out after an inactivity period to prevent long life session.
