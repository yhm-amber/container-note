[paper/arxiv]: https://arxiv.org/abs/2406.11717 "[Submitted on 17 Jun 2024 (v1), last revised 30 Oct 2024 (this version, v3)] { Machine Learning (cs.LG); Artificial Intelligence (cs.AI); Computation and Language (cs.CL) } (doi: 10.48550/arXiv.2406.11717) Refusal in Language Models Is Mediated by a Single Direction // 语言模型中的拒绝是由单一方向介导的 /// Conversational large language models are fine-tuned for both instruction-following and safety, resulting in models that obey benign requests but refuse harmful ones. While this refusal behavior is widespread across chat models, its underlying mechanisms remain poorly understood. In this work, we show that refusal is mediated by a one-dimensional subspace, across 13 popular open-source chat models up to 72B parameters in size. Specifically, for each model, we find a single direction such that erasing this direction from the model's residual stream activations prevents it from refusing harmful instructions, while adding this direction elicits refusal on even harmless instructions. Leveraging this insight, we propose a novel white-box jailbreak method that surgically disables refusal with minimal effect on other capabilities. Finally, we mechanistically analyze how adversarial suffixes suppress propagation of the refusal-mediating direction. Our findings underscore the brittleness of current safety fine-tuning methods. More broadly, our work showcases how an understanding of model internals can be leveraged to develop practical methods for controlling model behavior. // 对话式大语言模型在遵循指令和确保安全方面进行了微调，从而使得模型能够服从良性请求但拒绝有害请求。尽管这种拒绝行为在聊天模型中普遍存在，但其潜在机制仍不为人知。在这项工作中，我们表明拒绝行为是由一个一维子空间介导的，跨越了 13 个参数量高达 72B 的流行开源聊天模型。具体来说，对于每个模型，我们找到一个单一方向，使得从模型残差流激活中移除这个方向可以阻止其拒绝有害指令，而添加这个方向则会在无害指令上引发拒绝。利用这一见解，我们提出了一种新颖的透明箱越狱方法，该方法可以精确地禁用拒绝功能，同时极少影响其他能力。最后，我们通过机制分析，研究了对抗性后缀如何抑制拒绝介导方向的传播。我们的发现突出了当前安全微调方法的脆弱性。更广泛地说，我们的工作展示了如何利用对模型内部的理解来开发实际控制模型行为的手段。"
