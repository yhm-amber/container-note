[src/gh]: https://github.com/ORB-HD/deface.git "(MIT) (Languages: Python 98.2%, Shell 1.8%) Video anonymization by face detection // 通过人脸检测实现视频匿名化 /// deface is a simple command-line tool for automatic anonymization of faces in videos or photos. It works by first detecting all human faces in each video frame and then applying an anonymization filter (blurring or black boxes) on each detected face region. By default all audio tracks are discarded as well. // deface 是一个简单的命令行工具，用于自动匿名化视频或照片中的人脸。它的工作原理是首先检测每个视频帧中的所有人脸，然后对每个检测到的人脸区域应用匿名化滤镜（模糊或黑框）。默认情况下，所有音频轨道也会被丢弃。 /// How it works // 工作原理 /// The included face detection system is based on CenterFace (code (. gh: Star-Clouds/CenterFace.git), paper (. arxiv: 1911.03599)), a deep neural network optimized for fast but reliable detection of human faces in photos. The network was trained on the WIDER FACE (. web: shuoyang1213.me/WIDERFACE) dataset, which contains annotated photos showing faces in a wide variety of scales, poses and occlusions. // 该系统包含的人脸检测功能基于 CenterFace（ 代码 ， 论文 ），这是一个深度神经网络，专为快速可靠地检测照片中的人脸而优化。该网络使用 WIDER FACE 数据集进行训练，该数据集包含标注照片，展示了各种尺度、姿态和遮挡情况下的人脸。 /// Although the face detector is originally intended to be used for normal 2D images, deface can also use it to detect faces in video data by analyzing each video frame independently. The face bounding boxes predicted by the CenterFace detector are then used as masks to determine where to apply anonymization filters. // 尽管人脸检测器最初是为处理普通二维图像而设计的，但 deface 也可以通过独立分析每个视频帧来检测视频数据中的人脸。CenterFace 检测器预测的人脸边界框随后被用作掩码，以确定匿名化滤波器的应用位置。"
