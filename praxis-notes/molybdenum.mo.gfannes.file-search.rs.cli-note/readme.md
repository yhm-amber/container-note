[src/gh]: https://github.com/gfannes/molybdenum.git "(EUPL-1.1) (Languages: Rust 97.0%, Ruby 3.0%) Molybdenum Searcher /// Recursive, line-based search and replace CLI application. // 递归、基于行的搜索和替换 CLI 应用程序。 /// Basic usage // 基本用法 /// Following commands demonstrate how mo can be used to accomplish different tasks: // 以下命令展示了如何使用 mo 来完成不同的任务： /// - Create list of filenames, only filtering against the pathname itself: // 创建文件名列表，仅根据路径名本身进行过滤： /// - - mo: When no search pattern is specified, only the filenames are listed // mo : 未指定搜索模式时，仅列出文件名 /// - - mo -l: Explicitly ask to output only the filenames // mo -l : 明确要求仅输出文件名 /// - - mo -C FOLDER: Use FOLDER as root for searching // mo -C FOLDER : 使用 FOLDER 作为搜索根目录 /// - - mo -e hpp -e cpp: Only take files with hpp and cpp extension into account // mo -e hpp -e cpp : 仅考虑具有 hpp 和 cpp 扩展名的文件 /// - - mo -f PART: Keep filenames that match against PART // mo -f PART : 保留与 PART 匹配的文件名 /// - - mo -F PART: Keep filenames that do not match against PART // mo -F PART : 保留与 PART 不匹配的文件名 /// - - mo -0: Use 0x00 to separate filenames.
