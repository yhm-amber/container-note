Because the BLAS are efficient, portable, and widely available, they are commonly used in the development of high quality linear algebra software // BLAS（基本线性代数子程序）是一组提供执行基本向量矩阵运算的标准构建块的例程。第 1 级 BLAS 执行标量、向量和向量-向量运算，第 2 级 BLAS 执行矩阵-向量运算，第 3 级 BLAS 执行矩阵-矩阵运算。由于 BLAS 高效、可移植且广泛可用，它们在高质量线性代数软件的开发中得到了广泛应用。 /// LAPACK is written in Fortran 90 and provides routines for solving systems of simultaneous linear equations, least-squares solutions of linear systems of equations, eigenvalue problems, and singular value problems.
