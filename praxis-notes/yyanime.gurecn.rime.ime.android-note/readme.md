The name \"Yuyan\" combines the Chinese words for \"language\" (语) and \"swallow bird\" (燕), symbolizing agility and elegance in communication. Unlike many commercial input methods, YuyanIme focuses exclusively on providing an efficient typing experience while respecting user privacy through minimal permission requirements. // 语燕输入法（YuyanIme）是一款基于 Rime 引擎的开源安卓输入法编辑器。\"语燕\"这个名字结合了中文中\"语言\"（语）和\"燕子\"（燕）的字，象征着沟通中的敏捷与优雅。与许多商业输入法不同，语燕输入法专注于提供高效的打字体验，同时通过最小化权限要求来尊重用户隐私。 /// The project follows four core design principles: // 该项目遵循四个核心设计原则： /// - Pure input functionality - Focusing exclusively on text input without bloat // 纯粹的输入功能 —— 专注于文本输入，不添加冗余功能 /// - Minimal permissions - Only requesting permissions essential for operation // 最小化权限 —— 仅请求操作所必需的权限 /// - Rime-based but user-friendly - Making advanced input technology accessible // 基于 Rime 但用户友好——使高级输入技术易于使用 /// - Comprehensive input modes - Supporting various input methods and layouts // 全面的输入模式——支持多种输入方法和布局"
