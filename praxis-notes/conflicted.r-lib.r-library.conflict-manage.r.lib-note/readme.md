[src/gh]: https://github.com/r-lib/conflicted.git "(MIT) (Languages: R 100.0%) An alternative conflict resolution strategy for R // R 的另一种冲突解决策略 /// The goal of conflicted is to provide an alternative conflict resolution strategy. R’s default conflict resolution system gives precedence to the most recently loaded package. This can make it hard to detect conflicts, particularly when introduced by an update to an existing package. conflicted takes a different approach, making every conflict an error and forcing you to choose which function to use. // conflicted 的目标是提供一种替代的冲突解决策略。R 的默认冲突解决系统优先考虑最新加载的包。这可能会使检测冲突变得困难，特别是当冲突由现有包的更新引入时。conflicted 采取了不同的方法，将每个冲突视为错误，并迫使你选择要使用的函数。 /// Thanks to @krlmlr (. gh: @krlmlr) for this neat idea! This code was previously part of the experimental strict package, but I decided improved conflict resolution is useful by itself and worth its own package. // 感谢 @krlmlr 提出的这个巧妙想法！这段代码之前是实验性 strict 包的一部分，但我决定改进的冲突解决本身很有用，值得作为一个独立的包。"
