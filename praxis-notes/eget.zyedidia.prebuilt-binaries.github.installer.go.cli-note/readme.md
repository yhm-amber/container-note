[src/gh]: https://github.com/zyedidia/eget.git "(MIT) (Languages: Go 97.6%, Makefile 2.4%) Easily install prebuilt binaries from GitHub. // 轻松从 GitHub 安装预构建的二进制文件。 /// Eget is the best way to easily get pre-built binaries for your favorite tools. It downloads and extracts pre-built binaries from releases on GitHub. To use it, provide a repository and Eget will search through the assets from the latest release in an attempt to find a suitable prebuilt binary for your system. If one is found, the asset will be downloaded and Eget will extract the binary to the current directory. Eget should only be used for installing simple, static prebuilt binaries, where the extracted binary is all that is needed for installation. For more complex installation, you may use the --download-only option, and perform extraction manually. // Eget 是获取你喜爱工具的预构建二进制文件的最佳方式。它从 GitHub 的发布中下载并提取预构建的二进制文件。要使用它，请提供一个存储库，Eget 将搜索最新发布中的资源，试图找到适合你系统的预构建二进制文件。如果找到，该资源将被下载，Eget 将将二进制文件提取到当前目录。Eget 只应用于安装简单的、静态的预构建二进制文件，提取出的二进制文件是安装所需的一切。对于更复杂的安装，你可以使用 --download-only 选项，并手动执行提取。"
