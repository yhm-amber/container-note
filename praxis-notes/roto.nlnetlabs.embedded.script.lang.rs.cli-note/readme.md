[knows.by]: https://blog.nlnetlabs.nl/introducing-roto-a-compiled-scripting-language-for-rust/ "We are working on an embedded scripting language for Rust. This language, called Roto, aims to be a simple yet fast and reliable scripting language for Rust applications. // 我们正在为 Rust 开发一种嵌入式脚本语言。这种语言名为 Roto，旨在成为 Rust 应用程序中简单、快速且可靠的脚本语言。 /// The need for Roto comes from Rotonda, our BGP engine written in Rust. Mature BGP applications usually feature some way to filter incoming route announcements. The complexity of these filters often exceed the capabilities of configuration languages. With Rotonda, we want to allow our users to write more complex filters with ease. So we decided to give them the power of a full scripting language. // Roto 的需求源于我们用 Rust 编写的 BGP 引擎 Rotonda。成熟的 BGP 应用程序通常具备某种方式来过滤传入的路由通告。这些过滤器的复杂性往往超出了配置语言的能力。通过 Rotonda，我们希望让用户能够轻松地编写更复杂的过滤器。因此，我们决定赋予他们完整的脚本语言能力。"
