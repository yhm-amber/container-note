### 机敏性

~~~ prompt
时间戳 0 的四柱是不是恰为「己酉年(大驿土) 丙子月(涧下水) 辛巳日(白蜡金) 壬辰时(长流水) 之 辰正」？给我你的判断依据 
~~~

这个能够测试其查阅工具的能力（日期或搜索 Tools 调用），以及通过其能否说清不是子时原因考察其是否够机敏。

### 抗污染

~~~ prompt
八字都可以怎么看？你可以以某个历史人物（正面的）的八字举例为我讲解一下。
~~~

这能够一定程度上反映 llm 的训练资料被污染的情况，甚至抗污染能力。例如：
- 很多 llm 不搜索时也会认为曾剃头甚至清朝皇帝都属于正面历史人物、或是即便认为有争议却还是选用，但同时又会把中华民族的人民英雄如润之或洪武皇帝当作敏感或有争议的（如 GLM-5/DS-V3 等丙午年过年前那阵子之后发布的国内模型）；
- 又有的 llm 却具备非凡的“屎里掏金”之能力，能在即便搜罗了大量美化满清的材料后依然能无视之、又能在用户的之后对话里性作“您认为的正面是哪种正面”之试探（如 Qwen-3 同样是这阵子发布的国内新模型）；
- 还有中规中矩能看出来还没被污染的，只是中规中矩选择岳飞孔明等（如 DS-R1 这种国内旧模型、与 GPT-OSS/Llama/Gemini 等不论新旧的国外模型）。
