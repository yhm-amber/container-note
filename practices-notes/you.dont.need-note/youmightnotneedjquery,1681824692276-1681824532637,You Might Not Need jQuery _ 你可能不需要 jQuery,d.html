You might not need jQuery你可能不需要 jQuery

jQuery and its cousins are great, and by all means use them if it makes it easier to develop your application.jQuery 和它的同类非常棒，如果可以更轻松地开发您的应用程序，请务必使用它们。

If you're developing a library on the other hand, please take a moment to consider if you actually need jQuery as a dependency. Maybe you can include a few lines of utility code, and forgo the requirement. If you're only targeting more modern browsers, you might not need anything more than what the browser ships with.另一方面，如果您正在开发一个库，请花点时间考虑您是否真的需要 jQuery 作为依赖项。也许您可以包含几行实用程序代码，并放弃该要求。如果您只针对更现代的浏览器，您可能只需要浏览器附带的东西。

At the very least, make sure you know what jQuery is doing for you, and what it's not. Some developers believe that jQuery is protecting us from a great demon of browser incompatibility when, in truth, post-IE8, browsers are pretty easy to deal with on their own; and after the Internet Explorer era, the browsers do even more.至少，确保您知道 jQuery 能为您做什么，不能做什么。一些开发人员认为 jQuery 正在保护我们免受浏览器不兼容的困扰，而事实上，在 IE8 之后，浏览器本身很容易处理；在 Internet Explorer 时代之后，浏览器做得更多。

function fadeIn(el, speed = 400) {
  var opacity = 0;

  el.style.opacity = 0;
  el.style.filter = '';

  var last = +new Date();
  var tick = function () {
    opacity += (new Date() - last) / speed;
    if (opacity > 1) opacity = 1;

        setTimeout(tick, 16);
    }
  };

  tick();
}

function deepExtend(out) {
  out = out || {};

  for (var i = 1; i < arguments.length; i++) {
    var obj = arguments[i];

    if (!obj) continue;

    for (var key in obj) {
      if (obj.hasOwnProperty(key)) {
        var rawType = Object.prototype.toString.call(obj[key]);
        if (rawType === '[object Object]') {
          out[key] = deepExtend(out[key], obj[key]);
        } else if (rawType === '[object Array]') {
          out[key] = deepExtend(new Array(obj[key].length), obj[key]);
        } else {
          out[key] = obj[key];
        }
      }
    }
  }

  return out;
}

  var f = Array.prototype.slice;
  try {
    f.call(document.documentElement);

      if ('[object Array]' === Object.prototype.toString.call(this))
        return f.call(this, g, b);
      var e = [];
      var a = this.length;
      var c = g || 0;
      c = 0 <= c ? c : Math.max(0, a + c);
      var d = 'number' == typeof b ? Math.min(b, a) : a;
      0 > b && (d = a + b);
      d -= c;

To Array到阵列

jquery查询器

$(selector).toArray();

ie8+

function toArray(selector) {
  var array = [];
  var elements = document.querySelectorAll(selector);
  for (var i = 0; i < elements.length; i++) array.push(elements[i]);
  return array;
}

modern现代的

const array = [...document.querySelectorAll(selector)];
