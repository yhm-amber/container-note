OberonOberon

Oberon is a programming language in the same family as Pascal and Modula.Oberon 是一种编程语言，与 Pascal 和 Modula 同属一个家族。

Oberon is also the name of an operating system written in the Oberon language. Its main user-visible characteristic is an innovative "text user interface". From a technological point of view, Oberon's innovations include:Oberon 也是一个用 Oberon 语言编写的操作系统的名字。它的主要用户可见特征是一个创新的 "文本用户界面" 。从技术的角度来看， Oberon 的创新包括：

  • a "managed" runtime environment long before Java or C# came to market,早在 Java 或 C# 进入市场之前，就有了 "可管理" 的运行环境，
  • rock-solid type safety, permitting efficient multitasking entirely in kernel-space, and,坚如磐石的类型安全，允许完全在内核空间进行高效的多任务处理，以及，
  • a language whose specifications was actually smaller than K&R's original C specification, yet remains just as expressive, is fully semantically specified and proven (albeit by third parties), and easier to use.一种语言，其规范实际上比 K&R 的原始 C 规范要小，但仍然具有同样的表现力，完全是语义上的规范和证明（尽管是由第三方提供的），而且更容易使用。
